🗄️ DATABASE SCHEMA
====================

TABLE: persons (id, person_id, location, timestamp, image_path, 
                shirt_colors, pants_colors, hair_colors, confidence, 
                frame_index, video_source, notes, created_at, updated_at)
- UNIQUE INDEX: person_id
- INDEX: timestamp, location

TABLE: vehicles (id, vehicle_id, vehicle_type, license_plate, vehicle_colors,
                 location, timestamp, image_path, confidence, frame_index,
                 video_source, notes, created_at, updated_at)
- UNIQUE INDEX: vehicle_id, license_plate
- INDEX: vehicle_type, timestamp, location

TABLE: alerts (id, alert_type, person_id, vehicle_id, description, location,
               timestamp, frame_index, image_path, severity, status,
               created_at, updated_at)
- INDEX: alert_type, status, severity, timestamp
- FOREIGN KEY: person_id → persons.person_id
- FOREIGN KEY: vehicle_id → vehicles.vehicle_id


🚀 QUICK START
==============

1. Install PostgreSQL
   - Windows: https://www.postgresql.org/download/windows/
   - Docker: docker run --name postgres -e POSTGRES_PASSWORD=123456 -p 5432:5432 -d postgres

2. Install Dependencies
   pip install -r requirements.txt

3. Initialize Database
   python init_db.py

4. Start API Server
   python app.py

5. Test System
   python client_example.py

6. Integrate with main.py
   See INTEGRATION_GUIDE.py

🛠️ TROUBLESHOOTING
===================

PostgreSQL Connection:
  → Check .env DATABASE_URL
  → Ensure PostgreSQL service running
  → Verify credentials

Port Already in Use:
  → Change API_PORT in .env
  → Or kill process: lsof -i :5000

Module Not Found:
  → Run: pip install -r requirements.txt
  → Check Python environment

Database Exists:
  → Drop: python -c "from app import db; db.drop_all()"
  → Or: DROP DATABASE ai_detection; (in psql)
